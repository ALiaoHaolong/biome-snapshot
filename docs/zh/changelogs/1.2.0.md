新增：
* 增加 zh-hk（繁体中文中国香港）和 zh-tw（繁体中文中国台湾）的支持。
* 更新模组信息。
* 支持 modmenu。

修改：
* 图片导出路径更改为 `config/biome-snapshot/`，即保持与 MOD_ID 一致。
* 命令参数更名：mode->colorResolver，pos1->from，pos2->to。
* 优化代码。

移除：
* [1.18/1.18.2]移除点击文件名直接打开的功能（与 MC 高版本同步策略）。

修复：
* [1.18]所有基于 biome 的颜色解析器无法正常获取生物群系颜色。
