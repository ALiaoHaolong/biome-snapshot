sidebarTitle: 1.2.0

Added:
* Added support for zh-hk (Traditional Chinese, Hong Kong SAR) and zh-tw (Traditional Chinese, Taiwan).
* Updated mod information.
* Added Mod Menu support.

Changed:
* Changed image export path to `config/biome-snapshot/`, aligning with the MOD_ID.
* Renamed command arguments: mode -> colorResolver, pos1 -> from, pos2 -> to.
* Optimized code.

Removed:
* [1.18/1.18.2] Removed the feature that allows opening files directly by clicking the file name (aligned with higher Minecraft versions).

Fixed:
* [1.18] Fixed an issue where all biome-based color resolvers could not correctly retrieve biome colors.
