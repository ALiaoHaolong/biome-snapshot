Added:
* Added biome-snapshot-config command. This command can be used to configure color resolvers.

Changed:
* Unified colorResolver parameter values to lower snake case naming convention.
* Adjusted error handling logic for BiomeColorResolver.
* Changed Minecraft dependency version to >=, covering all stable versions between 1.18 and 1.21.4.
* Optimized code.
