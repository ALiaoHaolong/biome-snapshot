# BiomeSnapshot

一个用于 Minecraft 的模组，提供基于不同模式的生物群系快照功能，支持通过命令生成生物群系地图截图。

## 特性

- 支持多种生物群系颜色解析模式，包括默认模式、大陆与海洋模式、以及大陆、河流与海洋模式。
- 提供命令行接口，便于玩家或服务器管理员快速生成生物群系地图截图。
- 多语言支持，包括英文和中文。

## 使用方法

1. 安装 Minecraft Fabric 加载器。
2. 将本模组放入 `mods` 文件夹。
3. 启动游戏或服务器。
4. 在游戏中输入 `/biome_snapshot <模式>` 命令生成生物群系截图。支持的模式包括：
   - `default` - 使用默认颜色解析。
   - `mainland_ocean` - 使用大陆与海洋颜色解析。
   - `mainland_river_ocean` - 使用大陆、河流与海洋颜色解析。
